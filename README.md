# AIversity 新聞平台

一個現代化的 React.js 新聞平台樣板，提供豐富的互動功能和優雅的用戶體驗。

## 功能特色

### 1. 首頁（News List）
- 顯示多則新聞標題和預覽
- 響應式卡片設計
- 點擊標題導向詳情頁

### 2. 詳情頁（News Detail）
- **新聞類型選擇**：短新聞 vs 長新聞
- **文章版面配置**：
  - 左側（2/3 寬度）：新聞正文
  - 右側（1/3 寬度）：實時聊天室
- **名詞解釋功能**：點擊粗體詞彙顯示解釋
- **可拖曳調整寬度**：文章與聊天室區塊可左右拖曳

### 3. 互動功能
- 短篇/長篇切換按鈕
- 實時聊天室
- 名詞解釋工具提示
- 響應式設計

## 技術架構

- **React 18** - 前端框架
- **React Router** - 路由管理
- **Styled Components** - CSS-in-JS 樣式
- **React Hooks** - 狀態管理

## 專案結構

```
src/
├── components/
│   ├── NewsList.js          # 新聞列表頁面
│   ├── NewsDetail.js        # 新聞詳情頁面
│   ├── ChatRoom.js          # 聊天室元件
│   └── TermTooltip.js       # 名詞解釋工具提示
├── App.js                   # 主要應用元件
└── index.js                 # 應用入口點
```

## 安裝與運行

1. 安裝依賴：
```bash
npm install
```

2. 啟動開發伺服器：
```bash
npm start
```

3. 開啟瀏覽器訪問：http://localhost:3000

## 使用說明

1. **瀏覽新聞**：在首頁點擊任何新聞卡片進入詳情頁
2. **選擇新聞類型**：在詳情頁選擇「短新聞」或「長新聞」
3. **閱讀文章**：點擊粗體詞彙查看名詞解釋
4. **參與討論**：在右側聊天室與其他讀者交流
5. **調整版面**：拖曳分隔線調整文章和聊天室的寬度比例

## 核心功能實現

### 名詞解釋系統
- 使用正則表達式解析文章中的 `**粗體**` 標記
- 點擊觸發工具提示顯示解釋
- 支援 ESC 鍵關閉

### 可拖曳版面
- 使用 MouseEvent 監聽拖曳操作
- 動態調整 flex 比例
- 平滑的過渡動畫

### 實時聊天室
- 模擬即時訊息功能
- 自動滾動到最新訊息
- 支援 Enter 鍵發送

## 開發注意事項

- 所有樣式使用 Styled Components 實現
- 使用 React Hooks 管理狀態
- 響應式設計支援各種螢幕尺寸
- 模擬資料存放在各元件中，可輕易替換為 API 調用
