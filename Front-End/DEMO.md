# AIversity 新聞平台 - 功能演示指南

## 🎯 演示重點

本演示將展示 AIversity 新聞平台的核心功能，包括智能摘要、關鍵字註釋和全域 AI 聊天室。

## 🚀 快速體驗

### 1. 首頁佈局體驗
- **頂部導航**：查看品牌標識、導航菜單和用戶功能
- **焦點輪播**：自動輪播的重要新聞，支援手動控制
- **統一新聞卡片**：整合摘要、關鍵字註釋、相關推薦等功能

### 2. 統一智能新聞系統體驗
- **智能摘要**：點擊「展開」按鈕查看詳細內容
- **長短篇切換**：點擊「長篇模式」按鈕體驗不同閱讀模式
- **關鍵字註釋**：點擊橙色高亮的專業術語查看詳細解釋
- **相關新聞推薦**：展開後查看智能推薦的相關報導

### 3. 分類精選體驗
- **分類標籤**：點擊不同分類標籤（科技、金融、環境等）
- **即時篩選**：觀察內容如何根據分類動態變化
- **視覺反饋**：注意每個分類的專屬配色方案

### 4. 全域 AI 聊天室
- **浮動聊天**：查看頁面右下角的浮動聊天圖標
- **展開聊天室**：點擊聊天圖標展開聊天界面
- **選擇專家**：點擊專家標籤選擇想要諮詢的專家
- **發送訊息**：輸入問題或使用快速提示按鈕
- **多專家回覆**：觀察多位專家如何同時回答問題

## 🎨 設計亮點體驗

### 視覺效果
- **懸停動畫**：將滑鼠懸停在卡片上觀察上浮效果
- **漸變色彩**：欣賞各處使用的漸變背景
- **圓角設計**：注意統一的圓角設計語言
- **陰影效果**：觀察不同層級的陰影深度

### 響應式設計
- **調整視窗大小**：改變瀏覽器視窗大小觀察布局變化
- **側邊欄隱藏**：在小螢幕上側邊欄會自動隱藏
- **網格適配**：觀察內容網格如何自動調整

## 🔧 互動功能測試

### 統一新聞卡片功能
1. 點擊「展開」按鈕查看完整內容和相關新聞
2. 點擊「長篇模式」切換閱讀模式
3. 點擊「收藏」和「分享」按鈕
4. 查看閱讀統計數據
5. 點擊橙色高亮的關鍵字查看定義
6. 展開後查看智能推薦的相關報導

### 分類篩選功能
1. 點擊「科技」標籤查看科技相關新聞
2. 點擊「金融」標籤查看金融相關新聞
3. 點擊「環境」標籤查看環境相關新聞
4. 觀察每個分類的專屬配色

### AI 聊天室功能
1. 點擊右下角聊天圖標展開聊天室
2. 選擇 2-3 位不同領域的專家
3. 輸入問題如「最新科技趨勢是什麼？」
4. 觀察多位專家的回覆
5. 使用快速提示按鈕測試不同問題
